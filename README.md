# DietUnlocked Next.js App

Starter app for DietUnlocked.com, mirrored from a proven Next.js + Vercel + MDX workflow and reworked for the GLP-1 nutrition niche.

## Stack
- Next.js App Router
- MDX article content in `src/content/articles/`
- Vercel deployment
- GoDaddy-managed DNS pointing to Vercel

## Current status
This scaffold has been cleaned and rebranded into a DietUnlocked starter.
